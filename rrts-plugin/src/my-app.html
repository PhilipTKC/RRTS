Road Runner To Spreadsheet Extension

Spreadsheet ID found in URL

https://docs.google.com/spreadsheets/d/some-spreadsheet-id/edit#gid=0

Refresh Road Runner when saving Spreadsheet ID (First time only)

List of Defect Options

  • Ctrl + (Alt/Command) + ${$index - 1} Ctrl + (Alt/Command) + 0 Ctrl + (Alt/Command) + - Ctrl + (Alt/Command) + = ${defect} X

Refresh Road Runner when making changes to the defect list

The first 10 items are keybound to the following.

Ctrl + (Alt/Command) + 1 to 0 (Digits)

The next 2 items are keybound to the following.

Ctrl + (Alt/Command) + - (Minus) / = (Equals)
